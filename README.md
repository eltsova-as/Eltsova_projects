# AB-тестирование нового алгоритма рекомендаций в развлекательном приложении с «бесконечной» лентой
### Описание проекта
**Контекст:** Развлекательное приложение с функцией «бесконечной» ленты (аналог коротких видео) имеет две модели монетизации: платная подписка (без рекламы) и рекламная модель для пользователей без подписки. Команда разработки создала новый алгоритм рекомендаций, который предположительно показывает более интересный контент для каждого пользователя.

**Цель исследования:** Рассчитать параметры А/В-теста, оценить корректность его проведения и проанализировать результаты эксперимента для принятия решения о внедрении нового алгоритма. 

**Основные задачи:**
* Провести анализ исторических данных (EDA)
* Сформулировать гипотезы и определить ключевую метрику
* Рассчитать необходимый размер выборки и длительность теста
* Провести мониторинг первых дней эксперимента (проверка корректности распределения)
* Проанализировать результаты теста и оценить статистическую значимость
* Сформулировать рекомендации по внедрению

**Ключевая метрика:** Доля успешных сессий (просмотр 4+ страниц за сессию) – прокси-метрика удовлетворенности контентом и вовлеченности

**Стек:** Python (pandas, matplotlib, SciPy, StatsModels), Jupyter Notebook, A/B-тестирование, статистический анализ (z-тест, t-тест, Манна-Уитни)

**Topics:** Python, pandas, matplotlib, SciPy, StatsModels, A/B-тестирование, проверка гипотез, продуктовая аналитика, z-тест, размер выборки, MDE, power анализ, визуализация данных
